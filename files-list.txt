# 项目文件清单

## 必需文件（按优先级）

### 1. 核心代码文件
- main.go              # 主程序（最重要）
- go.mod               # Go模块定义
- go.sum               # 依赖锁定

### 2. 数据模型
- models/models.go     # 数据结构定义

### 3. 数据库文件（预留）
- database/database.go # 数据库连接

### 4. 文档文件
- README.md            # 项目文档
- API.md               # 接口文档

### 5. 可选文件
- server.log           # 运行日志（可忽略）

## 本地开发建议

1. 先复制核心文件 (main.go, go.mod)
2. 运行 `go mod tidy` 安装依赖
3. 测试本地运行 `go run main.go`
4. 逐步添加其他文件
